# 📊 E-Commerce Sales Dashboard (Excel Project)

This project is an interactive **E-Commerce Sales Dashboard** built entirely using **Microsoft Excel**.  
It includes two dashboards:

1. Sales Performance Dashboard 
2. Customer Behaviour & Segmentation Dashboard

These dashboards help visualize and analyze key business metrics such as:
- Total Sales
- Profit & Margin
- Regional Sales
- Product Category Insights
- Orders & Customer Segments
- Discounts, AOV, Shipping Time, etc.

## 📌 Project Highlights:

### 🔹 Sales Performance Dashboard
- Yearly & monthly sales trend  
- Sales by region & product category  
- Sales by channel  
- Top cities & top products  
- Revenue & profit by country  
- Discount vs Profit correlation  
- Interactive slicers (Year, Country, Category, Channel)

### 🔹 Customer Behaviour Dashboard
- New vs returning customers  
- Discount & shipping analysis by region  
- Country-wise sales map  
- Payment method usage  
- Average order margin  
- Preferred channel  
- Customer distribution by region & city  
- Top 10 customers by total spending

## 🎯 Purpose of the Project
To demonstrate how Excel can be used to create **professional BI-style dashboards** for analyzing e-commerce performance, customer behavior, and strategic business insights.

## 🚀 How to Use This Project
1. Download the Excel file. 
2. Open it in Microsoft Excel (Desktop version recommended).  
3. Interact with slicers to filter data by:
   - Year  
   - Country  
   - Region  
   - Product Category  
   - Channel  
   - Payment Method  
   - Shipping Days  
4. Explore insights across both dashboards.
